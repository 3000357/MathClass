GameTime Player
===============

GameTime Player is a frontend for libretro that auto detects cores.

HOW TO RUN
----------
Install [node](https://nodejs.org/).

From the command line:

`git clone https://github.com/matthewbauer/gametime-player.git`

`cd gametime-player`

`npm install`

`npm install -g grunt-cli`

`grunt run`
